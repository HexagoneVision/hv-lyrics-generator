# Security Policy

## Supported Versions

We actively maintain and support the latest stable version of **HV Lyrics Generator**. Please report any security issues affecting this or prior versions.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** create a public issue. Instead, contact us directly:

- Discord: [https://discord.gg/QQTnW4pqyu](https://discord.gg/QQTnW4pqyu)
- GitHub: [https://github.com/HexagoneVision/hv-lyrics-generator](https://github.com/HexagoneVision/hv-lyrics-generator)

We will respond within **72 hours** and work with you to resolve the issue responsibly.

## Security Best Practices

When using **HV Lyrics Generator**:

- Only use `.slrc` files from trusted sources. Malformed or malicious files could disrupt the conversion process.
- Always verify output LuaU scripts before running them in Roblox.
- Keep your Python environment and dependencies up-to-date.

## Supported Environment

- Python 3.x
- Linux / macOS / Windows (tested on Linux and macOS)

## Acknowledgements

We appreciate responsible disclosure of security vulnerabilities and will credit reporters in future releases if desired.
